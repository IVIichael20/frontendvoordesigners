README

De opdracht ging als volgt, maak een ontwerp dat werkt in een browser en device naar keuze. Doe dit door middel van een externe JSON file op te halen met Javascript. Zorg dat in je ontwerp meerdere UI events hebt waarmee je verschillende functionaliteiten kan uitvoeren. Hou ook rekening met verschillende interface principes en verschillende states uit de UI stack.

Zorg dat dit getest en uitgeschreven is en publiceer dit op GitHub.
